Instruções para o correto uso do software de caracterização de motores CC.


1 - Certifique-se de que todos os programas que serão usados estão nas versões compatíveis com as usadas no desenvolvimento.
2 - O programa para a análise de dados e integração software e hardware pode ser acessado através da plataforma QtCreator carregando-se o arquivo 'InterfaceGrafica.pro' na mesma. O uso do programa é intuitivo.

Configurações usadas no desenvolvimento:
    Ubuntu 18.04.3 LTS
    Qt version 5.9.5
    QMake version 3.1
    Qt Creator version 4.10.1
    Arduino version 1.8.10
    Fritzing version 0.9.3
